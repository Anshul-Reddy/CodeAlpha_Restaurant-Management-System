# 🍽 Restaurant Management System

A backend-based Restaurant Management System developed using **Django** as part of the **CodeAlpha Backend Development Internship**.

This application allows restaurant staff to efficiently manage menu items, table reservations, customer orders, and inventory through a simple web interface and Django Admin Panel.

---

##  Features

-  Dynamic Menu Management
-  Table Reservation System
-  Customer Order Management
-  Inventory Management
-  Automatic Table Availability Update
-  Automatic Inventory Update After Orders
-  Django Admin Panel for Data Management

---

## 🛠️ Technologies Used

- Python
- Django
- SQLite3
- HTML5
- CSS3

---

## 📂 Project Modules

###  Menu Management
- Display available menu items
- Manage menu through Django Admin

###  Table Reservation
- View available tables
- Reserve tables with customer details
- Automatically mark reserved tables as unavailable

###  Order Management
- Select reserved tables
- Place orders using multiple menu items
- Automatic total price calculation
- Order status tracking

###  Inventory Management
- Manage inventory for menu items
- Automatically reduce stock when an order is placed
